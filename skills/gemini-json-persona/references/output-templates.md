# Output Templates

Use these templates as writing scaffolds, not rigid schemas.

## 1. meta-moi.md

Recommended sections:

```md
# Meta Moi

## Niveau De Confiance
- `élevé`: ...
- `moyen`: ...
- `faible`: ...

## Identité De Travail
- ...

## Environnement Personnel
- ...

## Préférences Techniques
- ...

## Valeurs Et Sensibilités
- ...

## Style Cognitif
- ...

## Préférences De Collaboration Pour Les Agents
- ...

## Choses À Éviter Pour Les Agents
- ...

## Sources Significatives
- `conv_0001` ...

## À Confirmer Plus Tard
- ...
```

Guidance:

- This is the richest profile.
- Include confidence wording.
- Keep facts and interpretation distinct.

## 3. personal_telos_<name>.md

Recommended sections:

```md
# TELOS
## DOCUMENT PURPOSE
...

## HISTORY
...

## PROBLEMS
...

## MISSION
...

## NARRATIVES
...

## GOALS
...

## CHALLENGES
...

## STRATEGIES
...

## PROJECTS
...

## WISDOM
...

## METRICS
...

## LOG (Journal)
...
```

Guidance:

- Keep this reflective but grounded.
- Do not invent trauma, mental health diagnoses, or intimate biographical details from weak evidence.
- Use TELOS to express direction and meaning, not pseudo-certainty.
